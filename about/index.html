About Firebase Park

This application, "Firebase Park", is a demonstration of integrating Firebase services with a web front-end.

Developed by METSW, it showcases features like user authentication and real-time database interactions using Firestore for managing parking location data.

Core Technologies:

  • HTML5
  • CSS3
  • JavaScript
  • jQuery
  • Bootstrap 5
  • Firebase

Version 1.0.0 (PWA Focus)
